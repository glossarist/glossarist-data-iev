# glossarist-data-iev

IEC Electropedia (IEV) concept data for the Glossarist Vocabulary Browser.

This repository stores the harmonized IEV concept dataset as GCR packages, published as GitHub Release assets.

## Download

The latest GCR package is available from the [latest release](https://github.com/glossarist/glossarist-data-iev/releases/latest).
